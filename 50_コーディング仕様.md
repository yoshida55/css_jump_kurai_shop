# ストアサイト（書店） コーディング仕様

## 基本設定

| 項目 | 値 |
|------|-----|
| コンテンツ幅 | 最大1200px（横パディング5%） |
| ブレイクポイント | 896px |
| フォント | Arial, Hiragino Sans, Hiragino Kaku Gothic ProN, Meiryo, sans-serif |

---

## 各セクション仕様

### Header（メインビジュアル）
- 全画面表示（幅：100%、高さ：100vh）
- 背景画像を設定
- 高さはブラウザの高さにあわせて可変（`vh`単位を使用）
- 右上にロゴを配置（`position`を使用）

### Magazine / Archive / New
- セクション全体を`section`タグで囲む
- タイトルは`h2`タグ
- 画像は`flexbox`で横並び配置
- 画像の上に半透明ボックスを重ねる（`position`使用）
- 半透明は`background-color`の`rgba`で指定

### Fashion & Style
- セクション全体を`section`タグで囲む
- 全幅の背景画像を設定
- `background-size`で背景画像の表示サイズを指定
- 高さは固定値（ブラウザ幅変更時に変化しない）
- 背景画像の上にタイトル（`h2`）とボタン（`a`タグ）を配置

### Catalog / Antique
- セクション全体を`section`タグで囲む
- 背景にグレーを設定
- 画像とテキストを`flexbox`で横並び配置
- CatalogとAntiqueは共通CSSを使用 → `class`で命名（`.catalog-antique`）
- 画像を囲む`div`の横幅は`%`で指定（レスポンシブ対応）
- Antiqueにはボタン（`a`タグ）を配置

### Footer
- 全体を`footer`タグで囲む
- 背景カラーを設定
- ロゴとテキストのエリアを`flexbox`で3分割して横並び配置
- コピーライトは`p`タグで記述

---

## レイアウト構成

```
header          → 背景画像 + ロゴ（右上）
main
├── section#magazine    → タイトル + 画像（横並び・オーバーレイ）
├── section#fashion     → 背景画像 + タイトル + ボタン
├── section.catalog-antique（Catalog）→ 画像 + テキスト（横並び）
├── section.catalog-antique（Antique）→ 画像 + テキスト + ボタン（横並び）
footer          → ロゴ + テキスト3列 + コピーライト
```

---

## レスポンシブ対応

```css
/* PC */
@media (min-width: 897px) {
  /* PC用スタイル */
}

/* SP */
@media (max-width: 896px) {
  /* SP用スタイル */
}
```

## 技術ポイント

### 半透明オーバーレイ
```css
background-color: rgba(0, 0, 0, 0.5); /* 黒の50%透過 */
```

### 全画面表示
```css
height: 100vh;
width: 100%;
```

### 画像の横並び
```css
display: flex;
```

### 画像の上にテキストを重ねる
```css
position: relative; /* 親要素 */
position: absolute; /* 子要素（オーバーレイ） */
```

- [ ] メインビジュアルが全幅・全画面高さ（100vh）になっているか
- [ ] ロゴが右上に正しく配置されているか（position使用）
- [ ] Magazine/Archive/Newの画像が横並びになっているか
- [ ] 画像の上に半透明オーバーレイが正しく表示されているか
- [ ] Fashion背景画像が全幅で表示されているか
- [ ] Fashion背景画像の高さが固定になっているか
- [ ] Catalog/Antiqueの画像とテキストが横並びになっているか
- [ ] Footerが3分割で横並びになっているか
- [ ] 896px以下でSP表示に切り替わるか
- [ ] フォント指定が正しいか
