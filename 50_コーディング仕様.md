主要なコーディング仕様# コーポレートサイト コーディング仕様

## 基本設定

| 項目             | 値                                       |
| ---------------- | ---------------------------------------- |
| コンテンツ幅     | 1000px（メインビジュアル・About は全幅） |
| ブレイクポイント | 900px                                    |

---

## 各セクション仕様

### Header

- 右上「お問い合わせ」ボタン → SP で非表示（`display: none`）

### メインビジュアル

- 幅：全幅（100%）
- 高さ：画面下まで（`100vh`）

### セクションタイトル共通

- 英字タイトル + 日本語タイトルの 2 段構成
- 英字は `letter-spacing` で文字間隔を広げる

### News

- 3 カラム分割、間に罫線
- 日付は `<time>` タグを使用

### About

- 幅：全幅
- 画像高さ：400px 固定（`object-fit: cover`）

### Business

- 高さをずらした横並び配置
- SP 時の並び順に注意 → HTML の順番 or `order` プロパティで調整

### Company

- 幅：1000px
- 画像を上に重ねる（`position` 使用）
- 画像高さ：400px 固定

---

## レスポンシブ対応

```css
/* PC */
@media (min-width: 901px) {
  /* PC用スタイル */
}

/* SP */
@media (max-width: 900px) {
  /* SP用スタイル */
}
```

## チェックリスト

- [ ] コンテンツ幅 1000px になっているか
- [ ] メインビジュアルが全幅・画面高さになっているか
- [ ] News が 3 分割 + 罫線になっているか
- [ ] About が全幅・画像 400px になっているか
- [ ] Business の高さずれ・SP 並び順が正しいか
- [ ] Company の画像重ねが正しいか
- [ ] 900px 以下で SP 表示に切り替わるか
- [ ] SP でお問い合わせボタンが非表示か

ざっくりまとめ
ブレイクポイントの話
900px 以下でモバイル表示に切り替わる。
1000px は「コンテンツの最大幅」の話で、
ブレイクポイントとは別。
つまり：

1000px 以上 → コンテンツ幅は 1000px で中央寄せ
901px〜999px → まだ PC 表示（コンテンツは画面幅に合わせて縮む）
900px 以下 → モバイル表示に切り替え
